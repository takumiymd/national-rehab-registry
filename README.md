DESCRIPTION OF PROJECT

This project involves creating a database that tracks the wellbeing of 
addiction patients who are currently undergoing rehabilitation at a 
rehabilitation center.

The system is designed to support several applications, including:

Managing each patient based on their individual needs, such as tracking 
their progress in their recovery journey and monitoring the dosage of 
medication assigned to them.,

Providing useful data for Health Professionals, Researchers, Psychologists, 
and Academics. This data can help in identifying more effective methods 
for treating addiction and understanding which environments or approaches 
lead to better recovery outcomes.,


PROGRAMMING LANGUAGES USED:

NOTE: The database is private and only accessible to authorized healthcare-related professionals and experts.
